#### Port-Scanner

``
Java port scanner
``

#### Build

```bash
$ mvn clean package
```

#### Run

```bash
$ java -jar Port-Scanner-0.0.1-jar-with-dependencies.jar 192.168.1.1 80
or
$ java -jar Port-Scanner-0.0.1-jar-with-dependencies.jar example.com 1-500
or
$ java PortScanner example.com 1-5000
```
